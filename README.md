# Project 1, The Eight Puzzle
This is a Python program that solves the 8 puzzle using various search algorithms.

## Features
- Solving the project using the following search algoritms:
  - Uniform Cost Search
    A* Search with heuristic $h(n)$ is hardcoded to zero
- Solve the 8-puzzle problem using the following heuristics:
  - Misplaced Tile Heuristic
    A search that counts how many tiles are in the wrong position.
  - Manhattan Distance Heuristic
    A search based on the sum of the distances of the tiles from their goal positions.

## How to run
1. Prerequisites: Python 3.x installed.
2. Setup: Save the script as main.py.
3. Execution:
<pre> python3 main.py </pre>
